# 소리지갑 (Sound Wallet)

> 시각장애인을 위한 **AI 기반 한국 화폐 인식 음성 안내 시스템**  
> 스마트폰 카메라로 촬영한 **지폐·동전**을 인식하고, 금액을 **음성(TTS)** 으로 안내하는 접근성 서비스

---

## Overview

**소리지갑**은 시각장애인과 고령자처럼 시각 정보 접근이 어려운 사용자를 위해 만든  
**한국 화폐 인식 보조 서비스**입니다.

기존 화폐 인식 서비스는 외국 화폐 중심, 동전 미지원, 복잡한 UI, 온라인 의존성이 강한 경우가 많았습니다.  
소리지갑은 이러한 한계를 보완해 다음과 같은 방향으로 설계했습니다.

- **한국 지폐 + 동전 지원**
- **음성 중심 UI**
- **오프라인 사용 가능**
- **전맹 / 고령자 사용자까지 고려한 접근성 설계**

이 프로젝트는 단순한 객체 탐지를 넘어,  
실제 결제 상황에서 사용 가능한 **화폐 인식 → 금액 합산 → 음성 안내** 흐름을 구현하는 데 초점을 맞췄습니다. :contentReference[oaicite:1]{index=1}

---

## Why This Project?

시각장애인은 일상 속에서 다음과 같은 문제를 겪습니다.

- 동전은 촉각만으로 **구분이 어려움**
- 지폐는 마모되면 **촉각 인식이 어려움**
- 기존 앱은 외국 화폐 중심이거나 사용 방식이 복잡함
- 디지털 금융 서비스 접근성이 낮아 **현금 의존도**가 높음

소리지갑은 이러한 현실 문제를 해결하기 위해 시작한 프로젝트입니다.  
기술로 접근성을 높이고, **금전적 자립과 생활 편의 향상**에 기여하는 것을 목표로 했습니다. :contentReference[oaicite:2]{index=2}

## Key Features

- **한국 화폐 8종 인식**
  - 지폐: 1,000 / 5,000 / 10,000 / 50,000원
  - 동전: 10 / 50 / 100 / 500원
- **실시간 화폐 탐지 및 분류**
- **여러 화폐 금액 자동 합산**
- **한국어 음성 안내 (TTS)**
- **앱 / 웹 연동**
- **오프라인 사용 가능**

## System Flow

```text
스마트폰 카메라 입력
    ↓
YOLOv8n 기반 화폐 탐지/분류
    ↓
인식된 화폐 금액 합산
    ↓
TTS 음성 안내
    ↓
앱 / 웹 인터페이스 제공
